# Food Delivery Website ‚Äì SCSS Dynamic Architecture

A modern, responsive Food Delivery Website built with **HTML, SCSS (Modular Architecture), and JavaScript**.  
This project uses a **custom SCSS design engine** with variables, mixins, functions, spacing system, typography tokens, and responsive utilities.

## üìå Features

- Fully responsive UI using **SCSS utility engine**

- Reusable mixins (buttons, flex utilities, breakpoints)

  - Offers

  - App Download Section

  - Blog Section
